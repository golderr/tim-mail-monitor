# Web App

Next.js frontend for the Tim Mail Monitor internal dashboard.

## Current Scope

- Placeholder login screen
- Protected app shell
- Navigation stubs for the main staff workflows
- Clean, minimal layout for Milestone 1

## Run

```powershell
cd C:\real_estate_projects\tim-mail-monitor\apps\web
npm install
npm run dev
```

Visit `http://localhost:3000`.
